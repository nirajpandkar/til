# Augmented assignment implementation

For += -
```
>>> s = [1,2,3]
>>> t = [4,5,6]
>>> z = s
>>> s+=t
>>> s
[1, 2, 3, 4, 5, 6]
>>> z
[1, 2, 3, 4, 5, 6]
```
For concatenation -
```
>>> s = [1,2,3]
>>> t = [4,5,6]
>>> z = s
>>> s = s + t
>>> z
[1, 2, 3]
>>> s
[1, 2, 3, 4, 5, 6]
```
